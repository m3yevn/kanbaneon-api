Kanbaneon API
Web app Health GitHub

API Reference

Hapi.js + MongoDB · JWT auth · Jira-style boards, backlog, sprints, epics.

Base URL: https://kanbaneon-api.vercel.app

Auth

Most routes require Authorization: Bearer <token> from login or signup.

POST /api/v1/login   { "email", "password" }
POST /api/v1/signup  { "email", "password", "displayName" }
GET  /api/v1/profile
